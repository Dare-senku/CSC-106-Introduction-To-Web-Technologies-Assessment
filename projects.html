Project

CSC 106 Introduction to Web Technologies Lab Assessment

This project demonstrates the process of creating a personal portfolio web page using HTML, CSS and modern layout techniques. It focuses on structure, design, navigation and responsive presentation.

Overview

Lab Assessment Focus

The CSC 106 lab assessment is centered on building a functional web portfolio page. It includes separate pages for Home, About, Projects, Planner and Contact, with consistent navigation and a polished visual theme.

Process

Design and Implementation

The process began by planning the page layout and content for each section. HTML files were created for each page, navigation links were added, and CSS styles were used to make the interface clear, readable and visually appealing.

Skills

What I Learned

This assessment helped improve skills in semantic HTML, responsive design, navigation structure and visual hierarchy. It also reinforced how to present content clearly with page-specific sections and a consistent user experience.

Next Steps

Project Reflection

Future updates can include adding more interactive features, refining typography, and enhancing accessibility. The current page is a strong foundation for showcasing coursework and growing a personal portfolio.
